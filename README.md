# dadelutzifc.github.io

Personal academic homepage. Plain HTML/CSS, hosted on GitHub Pages.

## Editing

- **Content:** edit `index.html` — every spot you should change is marked with an `EDIT:` comment.
- **Look & feel:** edit `styles.css` (colors are CSS variables at the top).
- **Photo:** drop a square image named `profile.jpg` in this folder.
- **CV:** drop your `cv.pdf` in this folder (the header links to it).

## Preview locally

Open `index.html` in a browser, or run a tiny server:

```bash
python -m http.server 8000
# then visit http://localhost:8000
```

## Publish

This repo is named `dadelutzifc.github.io`, so once pushed to GitHub it serves at
**https://dadelutzifc.github.io**. Enable it under *Settings → Pages → Branch: main → /(root)*.
